# Changelog

All notable changes to AtlasConvert will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-07-01

### Added
- Interactive Atlas Map with civilization markers
- AnimatedNumber component for fluid conversion results
- Wiki articles for 10+ historical coins
- 7 new traditional measurement units

### Changed
- Upgraded AI model for improved performance
- Improved mobile chatbot with fullscreen mode
- Refined typography and vertical alignment

### Fixed
- Vercel API routing compatibility
- Chat scroll behavior during streaming
- Text alignment and font consistency

## [1.1.0] - 2026-05-15

### Added
- AI Chatbot with historical expertise
- Historical Playground (5 modes)
- API Playground for developers
- 8 premium Obsidian themes

### Changed
- Migrated AI infrastructure
- Improved offline support

## [1.0.0] - 2026-03-01

### Added
- Core currency converter (170+ currencies)
- Historical Museum (15 ancient coins)
- Traditional Regional Measures (17 units)
- Moroccan Colloquial engine
- PWA with offline support
- Trilingual interface (FR/EN/AR)
- Power Tools (custom units, scratchpad)
- Legal compliance pages
