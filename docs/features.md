## AI-Powered Assistant

A specialized numismatic and metrological AI that **never hallucinates** conversion values.

### How It Works

1. User asks a question in natural language
2. AI explains the historical context
3. AI calls the deterministic conversion engine via tool-calling
4. Accurate result is returned with context
