Overview

What is AtlasConvert?

AtlasConvert is an interactive museum and conversion engine for measurement systems across civilizations. It bridges the gap between modern currencies, historical coins, traditional regional units, and standard metric/imperial systems.

The Problem

Every day, historians, writers, students, and curious minds face the same challenge: understanding historical and traditional measurements. AI models systematically fail at these questions — they hallucinate conversion rates, confuse historical periods, and invent values. Standard converters handle only modern units. Wikipedia provides context but no direct answers.

The Solution

AtlasConvert provides a deterministic conversion engine combined with rich contextual data:

  • 15+ historical coins with academic purchasing power data
  • 17 traditional regional units across 7 civilizations
  • 170+ modern currencies with live exchange rates
  • AI-powered assistance that never hallucinates conversions
  • Trilingual interface (French, English, Arabic with RTL support)

AtlasConvert provides a public REST API for programmatic access to historical conversions, traditional units, and live exchange rates.

Base URL: https://atlas-convert.vercel.app/api

Authentication

API Key

Include your API key in one of three ways:

# Header (recommended)
curl -H "X-API-Key: ac_your_key_here" https://atlas-convert.vercel.app/api/currencies

# Bearer token
curl -H "Authorization: Bearer ac_your_key_here" https://atlas-convert.vercel.app/api/currencies

# Query parameter
curl "https://atlas-convert.vercel.app/api/currencies?apiKey=ac_your_key_here"

Conversion Engines

AtlasConvert uses three distinct conversion approaches:

  1. Standard Conversion — Factor-based multiplicative conversions for modern units
  2. Historical Conversion — Purchasing power and metal value calculations
  3. Traditional Conversion — Regional unit conversions via base unit intermediary

AI Integration

The AI assistant uses a tool-calling pattern:

User Query → AI Model → Tool Call → Conversion Engine → Response
                  ↓
         Historical Context
         (generated by AI)

The AI explains history; the engine computes numbers. This ensures zero hallucination on conversion values.

Security Model

  • Firebase Auth with Google OAuth + Email/Password
  • API key authentication for developer access
  • Role-based access control (free, premium, admin)
  • Rate limiting per API key tier
  • Input validation on all endpoints

Performance

  • Caching for exchange rates and metal prices
  • Edge deployment via Vercel CDN
  • PWA with offline caching
  • Code splitting for optimal load times

Why a Simple Converter Isn't Enough

A standard converter answers: "What is 1 USD in EUR?"

AtlasConvert answers: "What was 1 Roman Denarius worth to a legionary soldier in 100 AD, and how does that compare to a modern daily wage?"

The difference is context. Historical value isn't just a number — it's purchasing power, social status, and economic context. AtlasConvert provides all three.

Why a Simple Chatbot Isn't Enough

Ask ChatGPT: "How much was a Byzantine Solidus worth?"

Response: "A Byzantine Solidus was worth approximately $200 in modern currency."

Problems:

  1. The number is wrong (it was worth much more)
  2. No source cited
  3. No context about purchasing power
  4. No distinction between metal value and economic value

AtlasConvert's response:

  • Purchasing Power: ~$500 (a month's wages for a soldier)
  • Metal Value: ~$700 (7.3g of gold at current spot price)
  • Context: International trade standard for 1,000+ years
  • Sources: Goldberg (2007), Harper (2023)

The difference is accuracy, context, and citations.
